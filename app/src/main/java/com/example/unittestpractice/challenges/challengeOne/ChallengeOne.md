
# [CHALLENGE NUMBER 8]: (https://www.programiz.com/kotlin-programming/examples/vowel-consonant) 
- Check whether an alphabet is vowel or consonant

## Inputs/Outputs:
- Input -> a letter
- Output -> True

## Criteria:
- Should be a Letter

# POTENTIAL USE CASES:
## Fail:
- Is a Number
- Not a Vowel or Constant

## Success:
- A Vowel or Constant
- A Letter
- Not a Number
